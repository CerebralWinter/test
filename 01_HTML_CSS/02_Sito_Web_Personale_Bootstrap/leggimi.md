# landing page con Bootstrap HTML5 CSS custom FontAwesome ed Google Fonts

In questa versione del codice, ho incluso i collegamenti a Bootstrap CSS e JS nel tag head del documento.

Ho anche aggiunto classi di Bootstrap ai vari elementi per applicare le loro regole di stile e garantire una corretta struttura del layout.

L'intero menu di navigazione è stato implementato con le classi di Bootstrap per creare una barra di navigazione responsiva.

Le sezioni sono state avvolte in contenitori (div class="container") per garantire un layout responsive e uniforme.

Ho utilizzato classi di Bootstrap come py-5, text-center, bg-light, display-4, lead, e list-group per applicare stili appropriati alle varie sezioni e elementi della pagina.

Infine, ho incluso il collegamento a Bootstrap JS prima della chiusura del tag body per abilitare le funzionalità interattive di Bootstrap, come la navigazione responsiva nella barra
